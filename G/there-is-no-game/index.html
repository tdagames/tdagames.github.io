About the Game

There Is No Game is a hilariously meta puzzle adventure! The game insists there's nothing to play here, but you'll discover it's lying. Break the fourth wall, interact with the narrator, and solve creative puzzles in this unique gaming experience. Trust nothing - there IS a game!
